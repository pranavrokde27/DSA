		-->What is Linked List ?
			
			Linked list is a form of a sequential collection and it does not have to be in order. A linked list 
			is made up of independent nodes that may contain any type of data and each node to the next node in 
			the link.
-------------------------------------------------------------------------------------------------------------------		
		-->Array vs Linked List
		
		   -Element of linked list are independent objects.
		   -Variable size - the size of a linked list is not predefined.
		   -Random access -accessing an  element ia very efficient in arrays.
