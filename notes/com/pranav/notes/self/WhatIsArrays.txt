			--What is Arrays?
				
				- arrays can store data of specific type
				- elements of an array located in a contiguous.
				- each element of an array has a unique index.
				- the size of an array is predefined and cannot be modified.
			
			-- Defination of an array in computer science
			
				In computer science, an array is a data structure consisting of a collection of elements , 
				each identified by at least one array index or key. An array is such storeed that the position
				of each element can be computed from its index by a mathemmatical formula.
